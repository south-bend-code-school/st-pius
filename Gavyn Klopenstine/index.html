Build Your Own Biopage

Learn how to code by building your own website!

To build your own website, you’re going on a digital scavenger hunt. The clues below + Google will be the keys to having your own website.

Google anything that you don’t know!

Setup:

Follow the steps below to get your coding environment all set up! Here’s what you need:

Make a New Folder

  1. On the computer, go to “My Documents”
  2. Right-Click
  3. Select “Make a Folder”
  4. Name the folder your First Name and Last initial

Example: If my name is Camisa Vines, my folder would be named “Camisa V.”

IMPORTANT: All coding projects that you build will be saved in this folder.

Open Your Folder in Atom

We’ll be using a text editor called Atom as a place to write our code.

 

  1. Find Atom on your computer’s desktop

  1. Double click to open Atom

You should see a black screen that looks like this!

  1. Open the folder that you made in Atom (titled with your First Name and Last initial)

To open a folder, click File ---> Add Project Folder ---> Documents ---> Your Folder

Create New Files

Now that you’ve opened your folder in Atom, create a file to save in your folder.  

  1. File ---> New File
  2. Name the file “index.html”
  3. Click “Save”
  4. Create another file and name this one “style.css”

The two files:

index.html ←we’ll use this file to add text and images to our website

style.css ← use this to add colors and fonts to our website

Perfect! Your coding environment is now set up.

Start coding!
